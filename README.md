# Kaggle Practice Notebooks

Python basic applications, Data Manipulation, and Machine Learning exercises completed on Kaggle.

## Overview

This repository contains structured practice notebooks created on Kaggle. The notebooks cover Python fundamentals and transition into practical Data Analysis and Machine Learning model building.

## Topics Covered

**Python Fundamentals:**
* Python basics & Control flow
* Functions
* Data structures

**Data Manipulation (Pandas):**
* Creating, Reading, and Writing Data
* Indexing, Selecting, and Assigning
* Grouping and Sorting
* Data Types and Missing Values
* Renaming and Combining

**Intro to Machine Learning:**
* Building first ML models (Decision Trees, Random Forests)
* Model Validation (Train-test split, MAE)

**Intermediate Machine Learning:**
* Handling Missing Values (SimpleImputer)
* Categorical Variables (One-Hot & Ordinal Encoding)
* Scikit-Learn Pipelines
* XGBoost
* Preventing Data Leakage

## Purpose

To reinforce Python programming fundamentals and build a strong, practical foundation for Machine Learning and Data Science projects.
